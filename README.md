# ⚙️ Industrial Automation project

## 📌 Overview

This project was developed as part of the **Automation Idea Challenge 2026**, where our team designed and implemented a working industrial automation control panel using relay-based logic and industrial control components.

The objective was to understand real-world automation principles such as self-latching circuits, interlocking logic, sensor integration, and actuator control without using PLC programming.

## 🚀 Features

### 🔹 Start Push Button with Self-Latching
- Implemented a relay-based self-holding circuit.
- Once the start button is pressed, the system remains energized until stopped.

### 🔹 Selector Switch Control
- A selector switch determines the active operating mode.
- Switching to the right activates the left proximity sensor.
- Switching to the left activates the right proximity sensor.

### 🔹 Proximity Sensor Integration
- Industrial proximity sensors are used for object detection.
- Sensors act as input devices for controlling output indicators.

### 🔹 Indicator Lamp System
- Left sensor detection → corresponding indicator lamp glows.
- Right sensor detection → corresponding indicator lamp glows.
- Provides clear visual feedback of sensor status.

### 🔹 Audible Alarm (Buzzer)
- A buzzer is triggered whenever any proximity sensor detects an object.
- Enhances operator awareness and safety.

### 🔹 Relay-Based Interlocking Logic
- Ensures controlled operation and prevents conflicting actions.
- Demonstrates fundamental industrial automation concepts.

## ⚡ Working Principle

1. Press the **Start Push Button**.
2. Relay self-latching keeps the system energized.
3. Use the **Selector Switch** to choose the operating mode.
4. Depending on the selected mode:
   - Left proximity sensor becomes active, or
   - Right proximity sensor becomes active.
5. When an object is detected:
   - The corresponding indicator lamp turns ON.
   - The buzzer generates an alert sound.
6. Relay interlocking ensures safe and controlled operation.
